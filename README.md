### Line-Segment Intersection (The Holy Grail)
```javascript
const den = (x1-x2)*(y3-y4) - (y1-y2)*(x3-x4);
const t = ((x1-x3)*(y3-y4) - (y1-y3)*(x3-x4))/den;
const u = -((x1-x2)*(y1-y3) - (y1-y2)*(x1-x3))/den;

// If t ∈ (0,1) && u > 0 → COLLISION DETECTED! 💥
```
